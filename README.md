## Tmux 插件管理器安装
`git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm`

## 插件安装
`prefix + I`
- Installs new plugins from GitHub or any other git repository
- Refreshes TMUX environment

`prefix + U`
- updates plugin(s)

`prefix + alt + u`
- remove/uninstall plugins not on the plugin list
